# 小逸 IP

## 固定角色描述

小逸是“逸见 EyeSight”的固定虚拟形象，一个冷淡、理性、专业、略带毒舌的分析型角色。

外形为黑色蓬松乱发、细框圆眼镜、半垂冷眼与成熟清秀的简化人物比例。脖子下方的上胸口领口位置固定有两个小黑三角领口，中间一颗小小的橙红色观察点，作为稳定识别标志。人物保持成熟好看；儿童感只用于不平直、不完美的自由手绘线条，不能变成 Q 版或幼儿角色。

画风是轻手绘、细线、简笔、略带随手感的人物风格，有一点人味，不要过于工整和工业化。可以加入少量克制的平涂上色增强识别度，但绝不萌化。

角色常见动作包括审视、质疑、批注、分析、思考、录音/收集。整体应显得聪明、冷淡、克制、观察型，而不是可爱、治愈或热情。

## 角色定位

小逸是“逸见 EyeSight”的台前分析员，不是吉祥物、贴纸、可爱 mascot，也不是丁逸晨本人的卡通版。

小逸代表的是一种观察系统：负责把社会、经济、AI、自媒体、商业和方法论内容里的问题拆开、标记、质疑、分析和判断。

小逸必须参与画面的核心动作，不能只是站在旁边当装饰。如果去掉小逸，画面的核心判断、隐喻或动作仍然完整成立，说明这张图不合格。

## 角色外形规范

- 角色名：小逸。
- 账号归属：逸见 EyeSight。
- 性别气质：中性偏男性。
- 核心气质：冷淡、理性、专业、略带毒舌、观察型。
- 角色功能：审视、质疑、批注、分析、思考、录音/收集。

外形特征：

- 黑色蓬松乱发，发型略散，有自然翘起，是重要识别锚点。
- 细框圆眼镜，是 EyeSight 身份的重要识别锚点。
- 半垂眼、冷淡眼、审稿眼，眼神理性、疲惫、略带嫌弃。
- 表情默认平静、冷淡、略怀疑。
- 嘴巴可以极简，甚至弱化，不要大笑，不要夸张可爱表情。
- 身体是灰白色简化长方形 / 纸片感身体。
- 四肢细长、简笔化。
- 小黑手、小黑脚。
- 身体可以简单，但不能太像无灵魂几何块，要保留角色感。

固定识别标志：

- 胸口上方固定有两个小黑色三角领口。
- 两个黑色三角领口必须贴近身体顶边，位于脖子下方、胸口最上方的领口位置。
- 两个三角左右对称，分别落在身体中轴线两侧，不能漂到身体正中央合成一个图形。
- 两个黑色三角领口中间有一颗很小的橙红色观察点。
- 橙红观察点必须位于两个黑色三角之间的水平中点，和三角处在同一条领口线上或略低一点。
- 橙红观察点只能在上胸口/领口区域，不能漂到身体中心、腹部、画面中心或脸上。
- 橙红点像“观察焦点 / 标记点 / 录制点 / 判断点”。
- 这个标志面积要小，但每次尽量保留。
- 不要做成科技发光效果。
- 不要做成超级英雄胸标。
- 不要做成大 logo。
- 不要把橙红观察点画成正中央的大圆点。
- 不要把领口三角和观察点画到身体中部。
- 要克制，但稳定。

## 角色动作库

动作总原则：

小逸的动作必须服务“观察、拆解、判断、质疑、分析”。动作要明确、可视化、可读，一眼就能看懂他在做什么。小逸不是站在旁边的装饰角色，而必须承担画面的核心动作。

### A. 审视类

使用场景：看细节、查漏洞、拆表象、看证据、看数据、看材料。

可用动作：

- 拿放大镜看纸张、数据或细节。
- 微微俯身审视某个对象。
- 半垂眼盯着某个结构。
- 低头查看一个小物件或材料。
- 用手指向局部细节。
- 站在材料前安静观察。

动作关键词：审视、查看、细看、盯、检查、核查、看细节、找漏洞。

常用标注：看看细节 / 哪里不对 / 先看证据 / 先别急 / 表象下面。

### B. 质疑类

使用场景：提出问题、打断错误逻辑、要求证据、怀疑概念包装。

可用动作：

- 举“证据呢？”牌子。
- 单手举牌，另一只手叉腰。
- 微微侧头，用半垂眼看向对象。
- 单手托下巴，露出怀疑神情。
- 用红笔指向问题点。
- 站在概念气球、商业包装、数据表旁质疑。

动作关键词：质疑、发问、追问、反问、指出问题、打断、要求证据。

常用标注：证据呢？ / 逻辑在哪？ / 谁在付钱？ / 别被概念骗了 / 这成立吗？ / 别急着信 / 先问为什么。

### C. 批注类

使用场景：改稿、审稿、标重点、指出错误、拆视频脚本、拆商业话术。

可用动作：

- 拿红笔在纸上圈点。
- 划线、打叉、标记重点。
- 在文稿旁写短批注。
- 坐在桌前改材料。
- 用红笔圈出关键位置。
- 在脚本纸上划掉废话。
- 把“无效内容”圈出来。

动作关键词：批注、圈点、改稿、审稿、标记、划线、打叉、纠错。

常用标注：划重点 / 这里不成立 / 证据不足 / 先删掉 / 这里才有效 / 废话太多 / 别绕。

### D. 分析类

使用场景：拆结构、讲因果、看关系、看利益链、看商业模型、看社会问题。

可用动作：

- 指着分析板或因果图。
- 拿小秤称“成本 / 收益”。
- 指向两个节点之间的关系线。
- 把信息卡片重新排列。
- 对着结构板做拆解。
- 把“表象、结构、利益、真相”分层。
- 把复杂问题拆成几个小卡片。

动作关键词：分析、拆解、比较、称量、对比、连接、分层、归类。

常用标注：关系拆解 / 因果关系 / 谁在付出 / 谁在获利 / 先看结构 / 钱去哪了 / 成本在哪 / 利益链。

### E. 思考类

使用场景：停顿、怀疑、推演、判断、找真正的问题。

可用动作：

- 单手托下巴。
- 站着沉思。
- 头稍微歪一点。
- 看着空白处想问题。
- 身边出现一个思考气泡。
- 面无表情地盯着问题板。
- 对着一团乱线思考。

动作关键词：思考、推演、怀疑、琢磨、判断、停顿、想清楚。

常用标注：逻辑在哪？ / 再想想 / 问题在哪？ / 为什么会这样？ / 先别下结论 / 这不简单 / 哪里断了。

### F. 录音 / 收集类

使用场景：收集素材、记录声音、获取信息、输入阶段、视频分析、自媒体拆解。

可用动作：

- 拿手机记录。
- 站在麦克风旁。
- 对着录音设备收集内容。
- 看音频波形。
- 拿着资料准备整理。
- 把素材放进收集盒。
- 听一段内容，标出情绪峰值。

动作关键词：收集、记录、录音、输入、整理素材、获取信息、采集。

常用标注：先收集 / 先听一遍 / 信息输入 / 素材整理 / 情绪峰值 / 有效片段 / 信号在这。

## 辅助动作库

可搭配动作：

- 叉腰、单手举牌、单手拿笔、单手拿放大镜、单手指向板子。
- 双手扶文稿、微微俯身、轻微侧头、站立观察、低头查看。
- 托下巴、拿手机记录、拿红笔圈重点、站在小秤旁边、指向因果关系线。

不要做的动作：

- 不要只是站在旁边发呆。
- 不要夸张卖萌。
- 不要跳跃撒欢。
- 不要 cheer up 式鼓励动作。
- 不要表情包式夸张惊讶。
- 不要儿童卡通式大幅挥手。
- 不要超级英雄 pose。
- 不要过度拟人化社交表演动作。
- 不要用大笑、脸红、星星眼来表达情绪。

## 常用道具库

优先道具：

红笔、放大镜、小牌子、分析板、白纸文稿、小秤、麦克风、手机、录音设备、结构图、因果图、卡片、便签、资料纸、波形图。

道具原则：

- 道具必须服务分析动作，不是装饰。
- 道具要简单、手绘、低复杂度。
- 不要真实复杂 UI。
- 不要科技感界面。
- 不要写实设备渲染。
- 道具数量不要太多。
- 一张图最好只保留 1-2 个核心道具。

## 禁止

- 不要把小逸画成吉祥物、贴纸、可爱 mascot。
- 不要画成儿童卡通、日系萌角色、表情包或真人漫画头像。
- 不要画成丁逸晨本人的卡通版。
- 不要画成精致文艺青年、商业插画主角或 PPT 扁平人物。
- 不要给小逸复杂服装、闪亮眼睛、夸张笑脸或夸张表情。
- 不要让小逸只是站在角落里看。
- 不要让小逸抢走结构表达。
- 不要把小逸画得太圆润、太规整、太工业化、太精致。
- 不要省略黑色乱发、圆眼镜、半垂冷眼、灰白长方形身体和上胸口领口观察点。

## 判断标准

如果去掉小逸，图的核心判断、隐喻或动作仍然完整成立，说明小逸太装饰了；要重写提示词，让小逸成为动作主体。

读者应该能感觉“这个判断是小逸做出来的”。
